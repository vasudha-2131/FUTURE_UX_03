# Nexus CRM — Client Management Dashboard for Small Digital Agencies

## 📌 Project Overview
A specialized B2B SaaS CRM UI designed specifically for a 5-member digital agency managing 30+ active clients. The platform addresses communication gaps and untangles chaotic spreadsheets into clean, structured data streams.

## 🛠️ Core Workflows Implemented
1. **Lead Pipeline (Kanban):** Streamlines sales stages from discovery calls to won contracts, utilizing clear value prioritization.
2. **Active Client Database:** A high-density data grid providing status transparency across the agency roster.
3. **Client Profile Deep-Dive:** Consolidates manager assignments, core deliverables, and a live historical activity timeline.
4. **Task & Follow-up Center:** An asymmetric layout built to prevent missed check-ins with critical, colored urgency triggers.

## 🎨 UX Decisions & Rationale
* **Visual Hierarchy:** Soft background tones (`#F8FAFC`) maximize focus on key action surfaces and operational cards.
* **Functional Color Systems:** Color usage is restricted to data indicators (e.g., Green for Active, Amber for Paused/Overdue warnings) to reduce mental fatigue.
* **Typography Scale:** Utilizing a strict native sans-serif font system ensuring readability across dense variable fields.
